## Example Appium Test
Dieses Projekt ist eine beispielhafte Implementierung eines [Appium](https://appium.io) Tests, der in der Eco:Digit-Platform verwendet werden kann

### Bauen der Anwendung
Die Testanwendung kann mit dem folgenden Befehl gebaut und getaggt werden:
```shell
docker build -t eco_digit_appiumtest:latest .
```
### Ausführen der Anwendung

```shell
docker run --rm eco_digit_appiumtest:latest
```
