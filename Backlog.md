# Backlog – Potential Further Development

This backlog provides an overview of possible additional work packages that can be driven forward within the context of open-source development by communities.

---

## User Interface

Further adjustments to the user interface are conceivable:

- **Default Questions & Usage Hints:** A comprehensive concept for managing default questions or suggestions for using the test bench.
- **Progress Indicator:** A progress indicator during artifact uploads to keep users informed about the current status.
- Further development of the Help Bar

---

## Functional Enhancements

Several functional enhancements can be implemented to optimize measurement management:

- **Data Export:** The ability to export results at every display level as a CSV or PDF file.
- **LCA Tool Integration:** Embedding the LCA tool into the ECO:DIGIT user flow, so that the preparation of a measurement can be handled directly within ECO:DIGIT.
- **Artifact Validation:** Implementation of a validation mechanism for artifact uploads to verify the folder structure of the provided artifacts.

---

## Data Visualization

ECO:DIGIT offers various ways to display measurement results. This data visualization can be extended in the following areas:

- **Multipliers:** Addition of multipliers that allow measurement results to be projected onto larger time periods (days, weeks, months) or scaled to user numbers.
- **Measurement Contextualization:** When displaying a measurement, the underlying usage scenario could be shown again – e.g. number of clicks, users, or hardware considered – to help put the data into better perspective.
- **LLM Integration:** An AI-assisted interpretation of measurement results could help various domain stakeholders more easily understand the data and, for example, enable the automated generation of concise executive summaries.

---

## Deployment

- **SaaS Offering:** ECO:DIGIT could additionally be made available as a Software-as-a-Service solution.
