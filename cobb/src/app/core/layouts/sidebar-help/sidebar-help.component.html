This section provides a quick overview of the application's sustainability metrics over time. It highlights key trends and insights, helping you understand the application's current environmental impact.

Measurement

Here, you'll find detailed environmental impact data. Analyze trends, identify anomalies, and understand how the application behaves under different conditions. This data is essential for optimizing sustainability.

Artefact

The entirety of all files required to start a measurement, consisting of Usage Scenario, System Under Test, and Infrastructure Definition.

Application Context

An Application Context describes the context in which an overall system can be analyzed and evaluated. It encompasses various influencing factors and degrees of freedom that can affect the measurement – for example, different deployment models (e.g., on-premises or cloud), as well as additional technical framework conditions.

Infrastructure Definition

Formal description of the IT infrastructure and interfaces used in the real-world scenario.

Usage Scenario

A machine-readable definition of the interaction steps that are executed for the simulated usage of a system.
